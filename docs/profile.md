# Profile Management

This guide explains how to manage your user profile, preferences, and account settings in the Civo platform.

## Your Profile

Your profile contains your personal information and preferences that are visible to other club members.

### Viewing Your Profile

To access your profile:

1. **Click Profile Icon**: Located in the top navigation bar
2. **Select "My Profile"**: Navigate to your profile page
3. **View Information**: See your current profile details

Your profile displays:
- Name and contact information
- Email address
- Join date
- Club memberships
- Recent activity
- Profile picture (if set)

## Editing Your Profile

### Updating Personal Information

To modify your profile:

1. **Navigate to Profile**: Go to your profile page
2. **Click "Edit Profile"**: Start editing mode
3. **Update Information**:
   - **First Name**: Your first name
   - **Last Name**: Your last name
   - **Email**: Your email address
   - **Phone**: Optional phone number
   - **Bio**: Optional short biography
4. **Save Changes**: Click save to update your profile

!!! note "Email Changes"
    Changing your email address may require verification. Check your new email for a confirmation link.

### Profile Picture

Add or update your profile picture:

1. **Go to Edit Profile**: Navigate to profile editing
2. **Click "Upload Picture"**: Choose a profile picture
3. **Select Image**: Choose a file from your device
4. **Adjust**: Crop or adjust as needed
5. **Save**: Upload the new picture

**Image Guidelines**:
- Recommended size: 400x400 pixels
- Supported formats: JPG, PNG, GIF
- Maximum file size: 5MB
- Use clear, professional images

## Notification Preferences

Customize how you receive notifications:

### Email Notifications

Configure email notifications:

1. **Go to Settings**: Navigate to notification settings
2. **Email Preferences**: Choose what emails to receive:
   - **All Notifications**: Receive all emails
   - **Important Only**: Only critical notifications
   - **Daily Digest**: One summary email per day
   - **Weekly Summary**: One summary email per week
   - **None**: No email notifications

### Notification Types

Control specific notification types:

**Event Notifications**:
- New event invitations
- Event reminders
- Event changes or cancellations
- RSVP confirmations

**Club Notifications**:
- New news posts
- Important announcements
- Club invitations
- Join request updates

**Fine Notifications**:
- New fines issued
- Fine payment reminders
- Fine status updates

**Shift Notifications**:
- New shift assignments
- Shift reminders
- Shift changes

**Team Notifications**:
- Team assignments
- Team activities
- Team announcements

### Notification Timing

Set when to receive notifications:
- Immediate (as they happen)
- Hourly summary
- Daily digest (morning or evening)
- Weekly summary (choose day)

## Account Security

Manage your account security settings:

### Password Management

Change your password:

1. **Go to Security Settings**: Navigate to security section
2. **Click "Change Password"**: Start password change
3. **Enter Current Password**: Verify your identity
4. **Enter New Password**: Choose a strong password
5. **Confirm Password**: Re-enter new password
6. **Save**: Update your password

**Password Requirements**:
- Minimum 8 characters
- Include uppercase and lowercase letters
- Include at least one number
- Include at least one special character
- Don't reuse recent passwords

### Active Sessions

Monitor your active login sessions:

1. **View Active Sessions**: See where you're logged in
2. **Session Details**:
   - Device type and browser
   - Location (if available)
   - Last activity time
3. **Sign Out**: Log out of specific sessions
4. **Sign Out All**: Log out of all devices except current

### Two-Factor Authentication (If Available)

Add extra security to your account:

1. **Enable 2FA**: Navigate to security settings
2. **Choose Method**: SMS or authenticator app
3. **Verify Setup**: Complete verification
4. **Backup Codes**: Save recovery codes safely
5. **Activate**: Enable two-factor authentication

## Data and Privacy

### Downloading Your Data

Request a copy of your data:

1. **Go to Privacy Settings**: Navigate to data section
2. **Request Data Export**: Click to request your data
3. **Receive Email**: You'll receive a download link
4. **Download**: Get your complete data archive

Data export includes:
- Profile information
- Activity history
- Event participation
- Club memberships
- Messages and communications

### Data Retention

Understand data retention:
- Active account data is retained indefinitely
- Deleted data may be retained for backup purposes
- Logs and analytics follow platform policies

### Account Deletion

If you need to delete your account:

1. **Go to Account Settings**: Navigate to account section
2. **Request Deletion**: Click "Delete Account"
3. **Confirm Intent**: Verify you want to delete
4. **Processing**: Account marked for deletion
5. **Grace Period**: May have time to cancel

!!! warning "Account Deletion"
    Account deletion is permanent and cannot be undone. You'll lose access to all clubs and data.

## Troubleshooting

### Can't Update Profile?

- Ensure all required fields are filled
- Check that email format is valid
- Verify you're logged in
- Try refreshing the page

### Not Receiving Notifications?

- Check notification settings
- Verify email address is correct
- Check spam/junk folders
- Ensure notifications are enabled

### Can't Change Password?

- Verify current password is correct
- Check new password meets requirements
- Ensure passwords match
- Try clearing browser cache

### Profile Picture Not Uploading?

- Check file size (under 5MB)
- Verify file format (JPG, PNG, GIF)
- Try a different browser
- Ensure stable internet connection

### Settings Not Saving?

- Check internet connection
- Try refreshing the page
- Clear browser cache
- Contact support if issue persists

### Need Help?

Contact your system administrator for assistance with profile and account issues.
