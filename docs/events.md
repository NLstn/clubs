# Event Management

This guide covers how to create, manage, and participate in events within your club.

## Understanding Events

Events are scheduled activities that club members can attend. They can be:
- **One-time events**: Single occurrence events
- **Recurring events**: Events that repeat on a schedule (daily, weekly, monthly)

Each event can track:
- Event details (title, description, location)
- Date and time
- Member RSVPs
- Attendance tracking

## Viewing Events

### Upcoming Events

To view upcoming club events:

1. **Navigate to Your Club**: Select your club from the clubs list
2. **Click "Events"**: View all scheduled events
3. **View Event Details**: Click on an event to see full information

### Event Information

Event details include:
- Event title and description
- Date, time, and location
- RSVP list (who's attending, declined, or pending)
- Event organizer
- Number of attendees

## Creating Events (Administrators)

As a club administrator, you can create events:

1. **Open Admin Panel**: Go to your club's admin section
2. **Navigate to Events**: Select "Events" from the admin menu
3. **Click "Add Event"**: Start creating a new event
4. **Fill in Event Details**:
   - **Title**: Give your event a clear, descriptive name
   - **Description**: Provide details about the event
   - **Date and Time**: Set when the event will occur
   - **Location**: Specify where the event will take place
   - **Maximum Attendees** (optional): Limit the number of participants
5. **Save Event**: Click save to create the event

!!! success "Event Created"
    Your event is now visible to all club members who can RSVP.

### Creating Recurring Events

For events that repeat on a schedule:

1. **Follow Steps Above**: Start creating an event as normal
2. **Enable Recurrence**: Check the "Recurring Event" option
3. **Set Recurrence Pattern**:
   - **Frequency**: Choose daily, weekly, or monthly
   - **Interval**: Set how often (e.g., every 2 weeks)
   - **End Date**: Specify when the recurrence should stop, or set no end date
4. **Save**: The system will automatically create all recurring instances

!!! tip "Tip"
    Use recurring events for regular activities like weekly meetings, monthly gatherings, or daily training sessions.

## Managing Events (Administrators)

### Editing Events

To modify an event:

1. **Go to Admin Events**: Navigate to the events admin page
2. **Select Event**: Click on the event you want to edit
3. **Make Changes**: Update any event details
4. **Save Changes**: Don't forget to save your modifications

!!! note "Recurring Events"
    When editing a recurring event, you can choose to:
    - Update only this occurrence
    - Update this and all future occurrences
    - Update all occurrences

### Viewing RSVPs

Track who's attending your event:

1. **Open Event Details**: Navigate to the specific event
2. **View RSVP List**: See who has responded:
   - **Attending**: Members who confirmed they'll attend
   - **Declined**: Members who can't make it
   - **No Response**: Members who haven't responded yet
3. **Export List**: Download the attendee list if needed

### Canceling Events

If you need to cancel an event:

1. **Navigate to Event**: Go to the event in admin panel
2. **Select "Delete Event"**: Choose the delete option
3. **Notify Members**: Members with RSVPs will be notified of the cancellation
4. **Confirm**: Confirm the deletion

## Participating in Events (Members)

### RSVPing to Events

To respond to an event invitation:

1. **View Event**: Click on the event from the events list
2. **Choose Response**:
   - **Attend**: Confirm you'll be there
   - **Decline**: Indicate you can't attend
   - **Maybe**: Mark yourself as tentative (if available)
3. **Add Note** (optional): Include a message with your response

!!! note "RSVP Deadline"
    Some events may have RSVP deadlines. Respond before the deadline to ensure your spot.

### Changing Your RSVP

To update your response:

1. **Navigate to Event**: Go to the event details
2. **Update Response**: Change your RSVP status
3. **Save**: Your new response is recorded immediately

### Event Notifications

You'll receive notifications for:
- New events created
- Event reminders (before the event starts)
- Event updates or cancellations
- RSVP confirmations

## Event Calendar

### Viewing Your Schedule

See all your events in calendar view:

1. **Navigate to Dashboard**: Go to your main dashboard
2. **View "Upcoming Events"**: See events you've RSVP'd to
3. **Check Calendar**: View events by day, week, or month

### Filtering Events

Filter events to find what you're looking for:
- By date range
- By club (if you're in multiple clubs)
- By RSVP status
- By event type

## Best Practices

### Creating Effective Events

1. **Clear Titles**: Use descriptive titles that tell members what to expect
2. **Detailed Descriptions**: Include all relevant information (what to bring, dress code, etc.)
3. **Accurate Times**: Specify start and end times
4. **Location Details**: Provide clear directions or meeting points
5. **Lead Time**: Create events with enough advance notice

### Managing Event Attendance

1. **Send Reminders**: Remind members about upcoming events
2. **Follow Up**: Contact members who haven't RSVP'd
3. **Prepare Appropriately**: Use RSVP counts for planning
4. **Track Attendance**: Record actual attendance for future reference

### Recurring Events

1. **Regular Schedule**: Keep recurring events at consistent times
2. **Clear Pattern**: Make the recurrence pattern obvious (e.g., "Every Monday at 7 PM")
3. **Review Regularly**: Check if the recurrence pattern still works for your club
4. **Adjust as Needed**: Modify or stop recurrences that aren't working

## Troubleshooting

### Can't Create Events?

- Verify you have administrator privileges
- Check if there are any club settings restricting event creation
- Contact your system administrator

### RSVP Not Working?

- Ensure you're logged in
- Check if the event has reached maximum capacity
- Verify the RSVP deadline hasn't passed

### Not Receiving Event Notifications?

- Check your notification settings in your profile
- Verify your email address is correct
- Check spam/junk folders

### Event Not Appearing?

- Refresh the page
- Check the date filters
- Verify the event wasn't deleted or canceled

### Need Help?

Contact your club administrator or system administrator for assistance with event-related issues.
